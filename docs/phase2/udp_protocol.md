# UDP通信协议文档

## 基本信息
- **IP地址**: 127.0.0.1（本地测试）
- **端口**: 8888
- **协议**: UDP
- **数据格式**: JSON

## 手势ID映射表

| 手势ID | 手势名称 | 说明 |
|--------|----------|------|
| 0 | FIST | 握拳 |
| 1 | OPEN_PALM | 手掌张开 |
| 2 | POINT_INDEX | 食指指向 |
| 3 | VICTORY | 剪刀手 |
| 4 | THUMBS_UP | 点赞 |
| 5 | UNKNOWN | 未知 |

## 数据包格式

```json
{
    "hand_id": 0,           // 第几只手（0或1）
    "gesture_id": 0,        // 手势ID（0-5）
    "gesture_name": "FIST", // 手势名称
    "timestamp": 1647234567890  // 毫秒级时间戳
}
